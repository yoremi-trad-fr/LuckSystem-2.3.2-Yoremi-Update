Linux Release — AppImage
La version Linux est distribuée au format AppImage.

Le binaire lucksystem (patché Yoremi v3) et le dossier data sont embarqués directement dans l'AppImage — aucune installation requise.
Rendre le fichier exécutable avant le premier lancement :

  chmod +x LuckSystem-Yoremi-v3.AppImage

Double-cliquer ou lancer depuis le terminal :

  ./LuckSystem-Yoremi-v3.AppImage

Testé sur Linux Mint. Nécessite un environnement de bureau avec support WebKit2GTK.

Linux Release — AppImage
The Linux version is distributed in AppImage format.

The lucksystem binary (patched Yoremi v3) and the data folder are embedded directly in the AppImage — no installation required.
Make the file executable before launching it for the first time:

  chmod +x LuckSystem-Yoremi-v3.AppImage

Double-click or launch from the terminal:

  ./LuckSystem-Yoremi-v3.AppImage

Tested on Linux Mint. Requires a desktop environment with WebKit2GTK support.
